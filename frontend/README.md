# Pathfinder Frontend

Next.js app for the Pathfinder.io MVP.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
```

## Pages

- `/` landing foundation
- `/onboarding` path discovery form
- `/roadmaps` focused roadmap list

## API

The frontend reads `NEXT_PUBLIC_API_URL` from `.env.local`.

Default:

```text
http://localhost:4000/api
```
